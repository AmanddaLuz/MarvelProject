# MarvelProject

Nessa aplicação trabalhamos com: 

- Padrão de projeto: MVVM 
- Design Patterns 
- Injeção de dependências: Hilt 
- Layouts responsivos 
- Views Customizadas 
- API Rest 
- Persistência de dados 
- Android Jetpack (navigation, lifecycle etc...)
